# MindMaster 🧠

A comprehensive collection of **logical puzzle and strategy games** built with **Kotlin**. Challenge your mind with 10 different games, each featuring 3 difficulty levels (Easy, Normal, Hard).

## 🎮 Games Included

1. **2048** - Combine tiles to reach the 2048 tile
2. **Memory Match** - Test your memory by matching pairs
3. **Jigsaw Puzzle** - Complete picture puzzles piece by piece
4. **Water Sort** - Sort colored water into matching containers
5. **Sliding Tiles** - Arrange tiles in numerical order
6. **Sudoku** - Classic number puzzle game
7. **Math Quiz** - Solve mathematical problems under time pressure
8. **Word Guess** - Guess the word with limited attempts
9. **Quiz Game** - Test your general knowledge
10. **Odd Man Out** - Find the odd one in a group

## 🎯 Features

- **10 Unique Games** - Variety of logical and strategic puzzles
- **3 Difficulty Levels** - Easy, Normal, and Hard modes for each game
- **User-Friendly Interface** - Clean and intuitive design
- **Progressive Challenge** - Games scale in difficulty as you progress
- **Built with Kotlin** - Modern, reliable Android game development

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest version)
- Kotlin 1.5+ 
- Android SDK (API level 21 or higher)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aryan-Maurya-28/MindMaster.git
   cd MindMaster
   ```

2. **Open in Android Studio:**
   - Launch Android Studio
   - Select "Open an existing project"
   - Navigate to the cloned MindMaster folder
   - Wait for Gradle to sync

3. **Build and Run:**
   - Click the "Run" button or press `Shift + F10`
   - Select your target device/emulator
   - The app will install and launch

## 📱 Gameplay

Each game features:
- **Easy Mode** - Perfect for beginners and casual players
- **Normal Mode** - For intermediate players
- **Hard Mode** - Challenge for experienced puzzle solvers

Navigate between games from the main menu and select your difficulty level to start playing!

## 🎓 Game Descriptions

### Logic & Strategy Games
- **2048**: Slide and combine numbered tiles to create larger numbers and reach 2048
- **Sliding Tiles**: Rearrange shuffled tiles back to their correct numerical order
- **Jigsaw Puzzle**: Complete beautiful images by fitting puzzle pieces together

### Memory & Pattern Games
- **Memory Match**: Flip cards to find matching pairs and test your memory
- **Water Sort**: Pour colored water between containers to organize all colors
- **Odd Man Out**: Identify which item doesn't belong in a group

### Brain Teasers
- **Sudoku**: Fill the 9x9 grid following Sudoku rules
- **Math Quiz**: Solve arithmetic problems quickly and accurately
- **Word Guess**: Guess the mystery word with limited wrong attempts allowed
- **Quiz Game**: Answer trivia questions across various categories
